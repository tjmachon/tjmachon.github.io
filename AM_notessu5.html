1.5 Calculus of variations in higher dimensions

So far we have been thinking about one-dimensional motion. To really describe physics we want to look at more dimensions. In general, a particle will be described by a trajectory q(t) = (q1(t),q2(t),…,qn(t)). For example, in three dimensions q(t) = (x(t),y(t),z(t)). Our action S is then a functional of the full-three dimensional trajectory q(t), we write

S[q(t)] = S[(x(t),y(t),z(t))]

We say this is a functional of 3 dependent variables (x,y,z) and one independent variable t.

Proposition. A functional S[q(t)] of n dependent variables (q1(t),…qn(t)) is extremised if q(t) satisfies the n Euler-Lagrange equations

∂L ∂qi − d dt ∂L ∂q⋅i = 0

for each i = 1,2,…,n.

To establish this result, all you need to do is observe that the functional must be extremised for each function qi(t), these each give a separate Euler-Lagrange equation. This is sometimes written in compact notation as

∂L ∂q − d dt ∂L ∂q⋅ = 0

Now we can show how Hamilton’s principle gives Newton’s laws in higher dimensions. Our Lagrangian is

L = m 2 q⋅2 − V (q)

then we can check that

∂L ∂q = −∇V = F

where

∇V = ∂V ∂q1, ∂V ∂q2,…, ∂V ∂qn

and

∂L ∂q⋅ = mq⋅

so the Euler-Lagrange equations give Newton’s second law.

mq̈ = F
