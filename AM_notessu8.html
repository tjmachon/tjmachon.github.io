1.8 Constraints in the calculus of variations

1.8.1 Review: Lagrange multipliers

Suppose we want to minimise a function f(x,y) subject to the constraint g(x,y) = 0. The function g(x,y) = 0 can be viewed as a curve in the (x,y) plane. For example g(x,y) = x2 + y2 − 1, then g(x,y) = 0 is a circle. The vector field ∇g = (2x,2y) points normal (orthogonal) to this circle.

Now suppose we want to maximise f(x,y) subject to this constraint. Imagine sitting on the curve g(x,y) = 0, and measuring the value of f(x,y). If we have maximised f(x,y) on the curve g(x,y) = 0, then the change in f(x,y) must be zero if we move a little bit along the curve. Hence the gradient of f(x,y) must be orthogonal to the directions along the curve – hence it must be parallel to the normal of the curve g(x,y) = 0. But this normal is given by ∇g. So we find the condition ∇f = λ∇g.

Solving this equation for (x,y) then gives us equations in the unknown λ, which we pick to ensure that the constraint is satisfied. This works in any number of dimensions (setting ∇f = λ∇g).

Example. Maximise f(x,y) = x + y subject to the constraint g(x,y) = x2 + y2 − 1 = 0. Using Lagrange multipliers we find

∇f = λ∇g,(1,1) = λ(2x,2y)

solving for (x,y) gives

x = y = 1 2λ

but we know that x2 + y2 = 1, so

1 2λ2 = 1⇒λ = 1∕2.

Now we give the analogous problem in the calculus of variations. We have two functionals F[y(x)] and G[y(x)]. We would like to extremise F[y(x)] subject to the constraint that G[y(x)] = C = const.

Suppose y(x) is a solution, and consider a small variation y(x) → y(x) + αa(x) + βb(x) parameterised by the two constants α and β. Then we can think about F[y + αa + βb] and G[y + αa + βb] as functions in the (α,β) plane. But now this is a 2D constrained optimisation problem as a function of α and β, so we must be able to find a Lagrange multiplier such that

′ d dα(F − λG)α=β=0 = 0, d dβ(F − λG)α=β=0 = 0

but this must be true for all pairs of variations a and b. Hence we find the constrained Euler-Lagrange equation

δF δy − λδG δy = 0

and again λ is found by enforcing the constraint G[y] = C.

Example. The Catenary. Find the shape of an inextensible rope of constant density and length L, acted on by gravity, hung between two points A and B (which we can take to be at the same height), so A = (0,0), B = (b,0).

We describe our rope as a function y(x). The total energy of the rope, which we want to minimise, is purely potential, and is given by the height integrated over the entire rope:

V = ρg∫ 0Lydl,

where dl = dx2 + dy2. Assuming dx > 0 along the rope we can rewrite this as

V = ρg∫ 0by1 + (y′ )2dx

The constraint is that the rope must have total length L, so that

L = ∫ 0Ldl = ∫ 0b1 + (y′ )2dx

Hence our variational problem requires we extremise the functional

∫ 0b(ρgy − λ)1 + (y′ )2dx

This has no explicit x dependence, so the Beltrami identity applies and we find (assuming ρ = g = 1)

ρgy − λ 1 + (y′ )2 = c

for c constant. This can be solved to give

y(x) = λ + ccosh(x∕c + d)

for d a second constant. All three constants λ, c and d may be varied to find a solution that fits the boundary conditions. All hanging ropes, chains, etc. thus have the form of a cosh curve (called a ‘catenary’, from the latin for chain); the chains of a suspension bridge also have this form if the mass of the road beneath a suspension bridge can be effectively ignored.
