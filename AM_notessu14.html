2.5 Noether’s theorem

The Lagrangian formalism allows us to state one of the most profound results in all of theoretical physics, due to Emmy Noether.

Theorem. Noether’s theorem (informal). For every differential symmetry of a physical system there is a corresponding conservation law.

A simple example of a differentiable symmetry is a differentiable family of coordinate transformations on q that leave the action invariant, an example might be a rotation, or a translation. Let’s suppose we have a new family of generalised coordinates Q(q,t,𝜖), with the extra parameter 𝜖 describing the transformation. For example, suppose Q = (x′,y′) is given by a rotation of (x,y).

(x,y)↦(cos𝜃x − sin𝜃y,cos𝜃y + sin𝜃x)

then 𝜖 = 𝜃. We also require that for 𝜖 = 0 we recover the original coordinates, so

Q(q,t,0) = q

Then we have a family of actions depending on 𝜖 through Q.

S[Q] = ∫ tatb L(Q)dt

Definition. The transformation Q(q,t,𝜖) is a differentiable symmetry of a mechanical system with Lagrangian L if there is a function Λ(q,q⋅,t) such that

d d𝜖𝜖=0 ∫ tatb Ldt = ∫ tatb d dtΛ(q,q⋅,t)dt = Λ(q,q⋅,t)tatb

for all choices of endpoints ta < tb.

In other words, under the application of the symmetry the action changes by a total derivative. Now let’s look at the quantity

d d𝜖𝜖=0 ∫ tatb Ldt

applying the derivative yields

∫ tatb d d𝜖L𝜖=0 = ∫ tatb ∑ i ∂L ∂qigi + ∂2L ∂q⋅i∂g⋅i dt

where

gi = ∂Qi ∂𝜖 𝜖=0,g⋅i = ∂Q⋅i ∂𝜖 𝜖=0.

Now we know that the Euler-Lagrange equations are satisfied by q, so we use them to substitute for the first term to find

d d𝜖𝜖=0 ∫ tatb Ldt = ∫ tatb ∑ i d dt ∂L ∂q⋅igi + ∂L ∂q⋅ig⋅i dt

but this is a total derivative, so

d d𝜖𝜖=0 ∫ tatb Ldt = ∫ tatb d dt∑ i gi ∂L ∂q⋅i dt = ∑ i gi ∂L ∂q⋅i tatb

Now using the definition of differentiable symmetry we find

∑ i gi ∂L ∂q⋅i − Λtatb = 0

but since ta and tb were arbitrary this must be constant, and we arrive at Noether’s theorem.

Theorem. Noether’s theorem. For a Lagrangian with differentiable symmetry the quantity

∑ i gi ∂L ∂q⋅i − Λ

is a constant of the motion.

Now we study examples.

Example. Conservation of momentum for a single particle. Consider motion of a particle of mass m in 3 dimensions so q = (x,y,z), with no potential. Then the Lagrangian is just the kinetic energy.

L = m 2 (x⋅2 + y⋅2 + z⋅2)

Now consider the coordinate transformation generated along some the constant vector ex = (1,0,0), so that

Q(q,t,𝜖) = (x + 𝜖sx,y,z)

Under this transformation, the Lagrangian is invariant since Q⋅ is invariant. Since the Lagrangian is invariant, the function Λ must equal zero. Applying Noether’s theorem we find

d dt mx⋅ = 0

so that the x component of momentum is conserved. Applying the same to procedure to translations in the y and z directions we find that translation symmetry implies momentum conservation.

Example. Conservation of momentum in the two body problem. Consider the general two-body problem, so q = (x1,y1,z1,x2,y2,z2) with r1 = (x1,y1,z1) and r2 = (x2,y2,z2), with potential V (|r1 −r2|). Then the Lagrangian is

L = m1 2 r⋅1 2 + m2 2 r⋅2 2 − V (r 1 −r2).

Now consider a global translation along ex, so that

Q = (x1 + 𝜖,y1,z1,x2 + 𝜖,y2,z2)

this is a symmetry of the system, since it does not change the value of V or the velocities, and the Lagrangian is invariant, so Λ is invariant. Applying Noether’s theorem we find that

d dt m1x⋅1 + m2x⋅2 = 0,

which is the total momentum in the x direction. Again we can do the same for translations along y and z, and we find that

Translation symmetry ⇒ total momentum conservation

Example. Angular momentum conservation. Consider a 2D particle described by polar coordinates q = (r,𝜃) with a central potential V (r), then the Lagrangian is

L = m 2 r⋅2 + r2𝜃⋅2 − V (r).

We define the differentiable symmetry as a rotation, which just increases 𝜃 by a constant amount.

Q = (r,𝜃 + 𝜖)

The Lagrangian is invariant under this transformation, so Λ = 0. Applying Noether’s theorem yields

d dt mr2𝜃⋅ = 0

which is just angular momentum. Higher dimensional examples follow analogously and we have the general correspondence

Rotational symmety ⇒ angular momentum conservation

Example. Conservation of the Hamiltonian. Suppose the Lagrangian has no explicit dependence on t, so L = L(q,q⋅). Then we consider the ‘time translation’ symmetry, so that

Q(q(t),t,𝜖) = q(t + 𝜖)

Now under application of this symmetry we find

d d𝜖∫ tatb Ldt = ∫ tatb ∑ i ∂L ∂qiqi = ∂2L ∂q⋅i∂q⋅i dt = ∫ tatb dL dt dt

so that the function Λ = L, the Lagrangian, hence the time translation is a differentiable symmetry of the system. Now applying Noether’s theorem we find

d dt ∂L ∂q⋅ ⋅q⋅ − L = 0.

But recall that this is Beltrami’s identity, and that for a classical system, this quantity is just the total energy. Hence we have

Time translation symmety ⇒ total energy conservation
