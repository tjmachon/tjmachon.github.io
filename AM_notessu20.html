3.3 Poisson brackets

Definition. A (classical) observable of a Hamiltonian system is a function f(q,p,t).

Examples include angular momentum, total energy, distance of a particle from a target (or indeed any function you like). The time derivative of an observable is given by

d dtf = ∂f ∂t + ∂f ∂q dq dt + ∂f ∂p dp dt

which we rewrite as

d dtf = ∂f ∂t + ∂f ∂q ∂H ∂p −∂f ∂p ∂H ∂q

Definition. The Poisson bracket between two functions f and g on phase space is defined as

{f,g} = ∂f ∂q ∂g ∂p −∂f ∂p ∂g ∂q.

With this definition we may rewrite

d dtf = ∂f ∂t + {f,H}

The Poisson bracket satisfies several identifies

{f,g} = −{g,f},{f + g,h} = {f,h} + {g,h},{fg,h}} = f{g,h} + g{f,h}

and the Jacobi identity

{f,{g,h}} + {g,{h,f}} + {h,{f,g}} = 0

Note that we can consider our basic variables qi and pi as functions, which satisfy the simple relations (reminiscent of xp commutation relations in quantum mechanics)

{qi,pj} = δij,{pi,pj} = {qi,qj} = 0

and also the time evolution equations

d dtqi = {qi,H}, d dtpi = {pi,H}.

Using the Poisson bracket allows us to characterise conserved quantities in the following way, which we state without proof.

Proposition. A time-independent function f is conserved if and only if {f,H} = 0. Moreover, if f and g are conserved quantities, then {f,g} is also conserved.

The set of all conserved quantities then forms an algebra, the Poisson algebra.

Example. Poisson algebra of angular momentum. We consider a particle of mass m in three dimensions, q = (x,y,z), p = (px,py,pz) with potential V = V (r), where r2 = x2 + y2 + z2 = |q|2 is the radial coordinate. The Hamiltonian is

H = 1 2m p2 + V (r)

and the system conserves angular momentum

L = q ×p

To see this we compute

d dtL = q × d dtp = −q ×∇V + p ×p = −dV dr q ×q r = 0.

The three components of angular momentum are

Lx = ypz − zpy,Ly = zpx − xpz,Lz = xpy − ypx.

Now we can compute the Poisson bracket of these quantities and we find

{Lx,Ly} = Lz

and so on.

3.3.1 Canonical Quantisation

The connection between the Poisson brackets of classical mechanics and commutators of quantum mechanics suggests a route to quantisation of a given classical system, known as canonical quantisation. First proposed by Dirac, there are numerous problems with this approach. Canonical quantisation proposes to replace a Poisson algebra of observables with an algebra of operators where the Poisson bracket is replaced by the commutator. In short, if A and B are observables, and Â, B̂ the corresponding operators then we want

{A,B}↦ 1 iℏ[Â,B̂],

to truly make this work, one needs to develop deeper theories of quantisation.
