1.7 The brachistochrone and general problems in the calculus of variations

The toolbox we have developed can be used to do more than just derive Newton’s laws. Many problems in physics, engineering, maths (in fact science in general) can be phrased as optimisation problems for a functional. That is, we want to find a function x(t) that optimises some functional

ℒ = ∫ L(t,x,x⋅)dt

note that t may not be time, and x may not be space. The functional might also have more than one dependent variable, as in the previous section with x and y, or more than one independent variable4 e.g. t1 and t2. In all these cases, the optimal solution is found by solving the appropriate Euler-Lagrange equations. Let’s give an example, a problem originating with Johann Bernoulli.

Example. The Brachistochrone.

Find the form of the curve joining two fixed points A and B so that, starting from rest, a frictionless mass will slide down along it, under the influence of gravity alone, in the minimum amount of time.

Without loss of generality we make take A = (0,0) and B = (1,−h). We will describe the curve as a function y(x), and we have y(0) = 0 and y(1) = −h. For the problem to make physical sense, we require y(x) ≤ 0, otherwise the particle will never reach the end point (why?), this setup is shown in Figure 4. Now to formulate this as a calculus of variations problem, we need to compute the total time taken for the mass to slide from A to B.

The total time will be given as

T = ∫ ABdl v

where v is the velocity of the particle and dl2 = dx2 + dy2 is the infinitesimal length. Because the particle remains on the curve y(x) at all times, we have x = x(t) and y = y(x(t)), so assuming dx∕dt > 0, we may write dl as

dl = dx1 + y′ 2

where y′ = dy∕dx. Now total energy is conserved, and when x = 0 the height is zero, so we may write

1 2mv2 + mgy(x) = 0

hence we have

v = 2g−y(x)

which is well-defined since y(x) ≤ 0. Now we can write the total time as

T = 1 2g∫ 01dx1 + y′ 2 y

Then the Euler-Lagrange equations for T[y(x)] give

2y(x)y′′(x) + y′(x)2 + 1 −y(x)1 + y′ (x)2 = 0

now assuming y(x) < 0 for x > 0, this further simplifies to

2y(x)y′′(x) + y′(x)2 + 1 = 0

Now multiply by y′ to find

2y′yy′′ + (y′)3 + y′ = 0

Note that this can be rewritten as

d dx(y + y(y′)2) = 0

So we find the solution is

y + y(y′)2 = c

for some constant c. By separation of variables we find

y c − ydy = dx,

so that

x = a + ∫ y c − ydy

for some constant a. Now substitute y = csin2(−s), we find

x = a − 2c∫ sin2sds = a − cs + c 2sin2s

Now at y = 0 (so s = 0) we have x = 0, so a = 0. So our final solution is a curve given parameterically by

(x,y) = k(s −1 2sin2s,−sin2s)

where k = −c. This is the brachistochrone and, also known as a cycloid5. Note that it is independent of g, the brachistochrone is a solution for a particle on any planet in the universe. The value of k is determined by ensuring y = −h, when x = 0 (this is solved numerically).
