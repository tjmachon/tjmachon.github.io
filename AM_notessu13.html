2.4 D’Alembert’s principle of virtual work

This is an alternative development of the Euler-Lagrange equations. It can be useful for dealing with non-holonomic constraints.

Suppose we have M point particles, labelled by i with masses mi. Furthermore let ri be the Cartesian position vector of the ith particle. We say each particle is subject to a force Fitot = Fi + Ci, where Fi are motion generating forces, and Ci are constraint forces. Newton’s law then says

Fi + Ci − mir̈i = 0,i = 1,…,M

D’Alembert’s idea is to consider − mir̈i as a new inertial force which we write as Ii, then Newton’s law becomes

Fi + Ci − Ii = 0,i = 1,…,M

Now this is mathematically equivalent to the statement

∑ i=1N F i + Ci − Ii ⋅ δri = 0,forallδri

i.e. we may choose N infinitesimal displacements δri and the above equation must hold true for any such choice.

Recall that work done is equal to δW = F ⋅ δr, and in this way we can define the virtual work done δWV as

δWV = ∑ i=1N F i + Ci − Ii ⋅ δri

Proposition. D’Alembert’s principle. The infinitesimal virtual work done vanishes.

Proof. True by definition □

This is a generalisation of a principle of equilibrium. If a mechanical system is in equilibrium, then Ii = 0, and the virtual work becomes the real (infinitesimal) work.

This suggests a connection with calculus of variations, and Lagrangian mechanics. We say we have N generalised coordinates q = (q1,…qN), with ri = ri(q1,…,qN,t). We now write each of the three terms in the virtual work equation in terms of the qi.

1.
Motion forces. We have ∑ i=1MF i ⋅ δri = ∑ j=1N ∑ i=1MF i ⋅∂ri ∂qjδqj = ∑ j=1NQ jδqj.

2.
Constraint forces. These can be dealt with in the same way as inertial forces. However we can often choose our generalised coordinates cleverly so that the constraints are automatically satisfied (in fact this is done implicitly when we model the pendulum using an angle rather than (x,y) with a constraint on the length). If we make such a clever choice, the constraint forces vanish. We will assume this is the case.
3.
Inertial forces. We have ∑ i=1MI i ⋅ δri = ∑ j=1N ∑ i=1MI i ⋅∂ri ∂qjδqj = ∑ j=1NI jδqj

where Ij = ∑ iIi ⋅∂ri ∂qj. Now we know that I is related to the total kinetic energy K as Ii = −d(∂K ∂r⋅i)∕(dt) so that

Ij = −∑ i d dt ∂K ∂r⋅i ∂ri ∂qj = −d dt∑ i ∂K ∂r⋅i ∂ri ∂qj + ∑ i∂K ∂r⋅i d dt ∂ri ∂qj

The second term on the RHS can be written as

∑ i∂K ∂r⋅i d dt ∂ri ∂qj = ∑ i∂K ∂r⋅i ∂r⋅i ∂qj = ∂K ∂qj

The first term becomes

−d dt∑ i ∂K ∂r⋅i ∂ri ∂qj = −d dt∑ k=1N ∑ i=1M ∂K ∂q⋅k ∂q⋅k ∂r⋅i ∂ri ∂qj

Now ∂q⋅k ∂r⋅i = ∂qk ∂ri so this becomes

−d dt∑ k=1N ∑ i=1M ∂K ∂q⋅k ∂qk ∂ri ∂ri ∂qj = −d dt∑ k=1N ∂K ∂q⋅k ∂qk ∂qj = −d dt ∂K ∂q⋅j

where we use ∂qk ∂qj = δkj. Putting it all together we find

Ij = −d dt ∂K ∂q⋅j + ∂K ∂qj.

The result of this is that D’Alembert’s principle becomes

∑ j=1N Q j + ∂K ∂qj − d dt ∂K ∂q⋅j δqj = 0.

Definition. A force Qj is monogenic if there is a generalised potential V (q,q⋅,t) such that

Qj = −∂V ∂qj + d dt ∂V ∂q⋅j

If we assume a monogenic force, D’Alembert’s principle becomes

∑ j=1N ∂L ∂qj − d dt ∂L ∂q⋅j δqj = 0.

where L = K − V is the Lagrangian. Since δqj is arbitrary, this implies the Euler-Lagrange equations.
