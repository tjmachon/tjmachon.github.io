3.4 Canonical transformations

One of the key strengths of the Lagrangian approach is that we may use any set of generalised coordinates that we like in order to solve a problem, the Euler-Lagrange equations still work in those coordinates. In Hamiltonian mechanics we can have even more transformations! We can mix q and p if we like, as long as Hamilton’s equations still have the same form.

Definition. A coordinate transformation (q,p)↦(Q,P) that preserves the structure of Hamilton’s equations is a canonical transformation.

This means that the new coordinates satisfy

d dtQ = ∂H ∂P, d dtP = −∂H ∂Q.

If the coordinate transformation is time-independent, then there is a very geometric characterisation of canonical transformations.

Proposition. Let Q(q,p) and P(q,p) be a new set of coordinates. Define the Jacobian matrix as a block matrix

J = JQqJQp JPqJPp

where

(JQq)ij = ∂Qi ∂qj ,(JQp)ij = ∂Qi ∂pj

and so on. Then the coordinate transformation is canonical if and only if

JT ΩJ = Ω,

where

Ω = 0 I −I 0

is the symplectic form.

Proof. The time derivative of Q and P is given by (using the chain rule):

d dt Q P = J⋅d dt q p

The using Hamilton’s equations for q and p we find

d dt Q P = J⋅Ω⋅∂H ∂q ∂H ∂p

Now, using the chain rule we find

∂H ∂q ∂H ∂p = JT ⋅∂H ∂Q ∂H ∂P

so we find

d dt Q P = J⋅Ω⋅JT ⋅∂H ∂Q ∂H ∂P .

But this is equal to Hamilton’s equations for Q and P if and only if

J ⋅ Ω ⋅JT = Ω⇔JT ⋅ Ω ⋅J = Ω

□

The condition JT ⋅ Ω ⋅J = Ω must hold at all points in phase space. Any matrix J satisfying this condition is a 2N dimensional symplectic matrix. We can expand out the conditions by rewriting JT ⋅ Ω ⋅J = Ω as JT ⋅ Ω = Ω ⋅J−1. This can then be written out as explicit conditions for a canonical transformation

∂Pi ∂pj = ∂qj ∂Qi, ∂pj ∂Qi = −∂Pi ∂qj , ∂qj ∂Pi = −∂Qi ∂pj ,∂Qi ∂qj = ∂pj ∂Pi

which is in fact just two conditions

∂Pi ∂pj = ∂qj ∂Qi, ∂pj ∂Qi = −∂Pi ∂qj

Example. Cartesian to polar canonical transformation in 2D. Suppose we have a point particle in two dimensions, with Hamiltonian

H = 1 2m(px2 + p y2) + V (x,y)

where px = mx⋅, py = my⋅. Defining polar coordinates

r = x2 + y2,𝜃 = arctan y x,x = rcos𝜃,y = rsin𝜃

x⋅ = r⋅cosϕ − rϕ⋅sinϕ,y⋅ = r⋅sinϕ + rϕ⋅cosϕ

In polar coordinates the Hamiltonian becomes

H = 1 2m pr2 + p𝜃2 r2 + V (r,𝜃)

where

pr = xpx + ypy x2 + y2 ,p𝜃 = xpy − ypx

This allows us to compute the Jacobian matrix

J = x∕r y∕r 0 0 −y∕r2 x∕r2 0 0 −yp𝜃∕r3xp𝜃∕r3x∕ry∕r py −px −y x

and you can check that JT ⋅ Ω ⋅J = Ω, so that the transformation is canonical.

Example. Time translation. An interesting canonical transformation is time translation. For this we define the new coordinates

Q(t) = q(t + τ),P(t) = p(t + τ)

for some fixed time τ. This gives us

∂Qi ∂pj = 0 = ∂pi ∂Qj

the first of our conditions for a canonical transformation. We also write our coordinates as as

Q(t) = ∫ 0τq⋅(t + τ′)dτ′,P(t) = ∫ 0τp⋅(t + τ′)dτ′

Now we have elements of the Jacobian matrix

∂Qi ∂qj = ∫ 0τ ∂2H ∂pi∂qjdτ′

∂Pi ∂pj = −∫ 0τ ∂2H ∂qi∂pjdτ′ = ∂qi ∂Qj

which gives the second the condition for a canonical transformation.

Example. Time translation for the 1D harmonic oscillator. The 1D harmonic oscillator has Hamiltonian

H = 1 2mp2 + k 2q2

with Hamilton’s equations

q⋅ = p m,p⋅ = −kq

with solutions q = q0 cosωt + p0 sinωt, p = p0 cosωt − q0 sinωt for initial conditions (q(t = 0),p(t = 0)) = (q0,p0) and frequency ω = k∕m. If we define our time translated coordinates

(Q(t),P(t)) = (q(t + τ),p(t + τ))

we find

Q(t) = q0 cosω(t + τ) + p0 sinω(t + τ) = qcosωτ + psinωτ

P(t) = pcosωτ − qsinωτ

and the 2 × 2 Jacobian is

J = cosωτ,sinωτ −sinωτ,cosωtau

It is easy to check that J is a symplectic matrix, so the transformation is canonical.

Symplectic matrices are very special, they define a type of geometry (symplectic geometry), different from the geometry of space (Euclidean geometry) and spacetime (Lorentzian geometry). In fact, all three types of geometry can be described by the types of transformations that preserve a given structure. In this way classical mechanics is essentially “geometry in phase space” (VI Arnold).

Euclidean Geometry
Linear transformations that preserve the dot product in Euclidean space, in 2D this is x1x2 + y1y2. This implies we seek transformation matrices O so that OT IO = I,I = 10 0 1 .

This defines orthogonal transformations (rotations and reflections).

Lorentzian Geometry
Linear transformations that preserve the ‘Lorentz product’ (in 2D with c = 1 this is x1x2 − t1t2). This implies we seek transformation matrices Λ so that ΛT ηΛ = η,η = 1 0 0 −1 .

This defines Lorentz transformations.

Symplectic Geometry
Linear transformations that preserve the product q1p2 − p1q2. This implies we seek transformation matrices J so that JT ΩJ = Ω,Ω = 0 1 −1 0 .

This defines symplectic transformations.
