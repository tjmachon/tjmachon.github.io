1.6 Special cases and the Beltrami identity

The Euler-Lagrange equations are second-order, and hence typically difficult to solve. In many cases, however, they can be simplified considerably. We consider a functional

F[x(t)] = ∫ 0T f(t,x,x⋅)dt

and three special cases.

1.
f(t,x,x⋅) = f(t,x), no explicit x⋅ dependence. The Euler-Lagrange equations give the equation ∂f ∂x = 0.

2.
f(t,x,x⋅) = f(t,x⋅), no explicit x dependence. In this case we see that d dt ∂f ∂x⋅ = 0

hence we find the first-order differential equation

∂f ∂x⋅ = const.

3.
f(t,x,x⋅) = f(x,x⋅), no explicit dependence on t. This is often the case in physics. Consider the total derivative of f with respect to t, we have df dt = ∂f ∂t + ∂f ∂xx⋅ + ∂f ∂x⋅ẍ = ∂f ∂xx⋅ + ∂f ∂x⋅ẍ

Now apply the Euler-Lagrange equations to the first term to find

df dt = x⋅ d dt ∂f ∂x⋅ + ∂f ∂x⋅ẍ = d dt x⋅∂f ∂x⋅

hence we find

d dt f −x⋅∂f ∂x⋅ = 0

This implies the Beltrami identity.

Definition. The Beltrami Identity is

f −x⋅∂f ∂x⋅ = const.

This identity is very important (we will see later). In multiple dimensions it becomes

f −q⋅ ⋅∂f ∂q⋅ = const.

where we use the dot or scalar product of vectors. The Beltrami identity often allows us to convert the second-order Euler-Lagrange equations into first order equations, which can be solved much more easily. Let’s think about the Beltrami identity when we have a Lagrangian of the form

L = T(q⋅) − V (q).

In this case we find

T − V −q⋅ ⋅∂T ∂q⋅ = const.

Now let’s assume that T = m|q⋅|2∕2, then we find

−−m|q⋅|2 2 + V + mq⋅ ⋅q⋅ = −(T + V ) = const.

this is just conservation of the total energy T + V .

4.
f is a total derivative of a function g(t,x) so f = d dtg(t,x),

then we find that the functional is

F = ∫ 0T dtf = [g(t,x)] 0(T,xT)

since x(T) = xT is a fixed boundary condition, this functional has the same value for all functions x(t) that satisfy the boundary conditions, so the Euler-Lagrange equations are trivial. This example shows that two different functionals can give the same Euler-Lagrange equations.
