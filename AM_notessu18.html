3.1 The Beltrami identity and Legendre transforms

We now study a third formulation of mechanics, the Hamiltonian formulation. This is the deepest formulation of classical mechanics, recasting classical mechanics as geometric problems in ‘phase space’. It is often less suited to direct computation than the Lagrangian formulation.

For a Lagrangian system with Lagrangian L that does not explicitly depend on time t, we found that the quantity

H = ∂L ∂q⋅ ⋅q⋅ − L

was a constant of the motion. Explicit computation showed that this was related to the total energy of the system, a result we also derived using Noether’s theorem. This quantity is known as H, the Hamiltonian, and is the Legendre transform of the Lagrangian.

Definition. A function f(x) is convex if

f((1 − t)x0 + tx1) ≤ (1 − t)f(x0) + tf(x0)

if f is twice-differentiable, this is equivalent to the condition that

d2f dx2 ≥ 0.

Definition. A function f(x) is strictly convex if

f((1 − t)x0 + tx1) < (1 − t)f(x0) + tf(x0)

if f is twice-differentiable, this is equivalent to the condition that

d2f dx2 > 0.

Definition. The Legendre transformation of a strictly convex function f(x) is the function f∗ of p given by

f∗(p) = max x xp − f(x)).

This just means ‘pick a value of p, then find the maximum value of the quantity xp − f(x) over all values of x, the maximum value is the Legendre transform’. If f is differentiable, then xp − f(x) will be maximised if

∂ ∂x = 0 = p − f′(x)⇒p = f′(x).

If the function is differentiable, then the Legendre transform has a nice graphical interpretation, shown in Figure 25.

Proposition. The Hamiltonian H(q,p) is the Legendre transformation of the Lagrangian L(q,q⋅,t) with respect to q⋅.

Proof. Recall the Hamiltonian is defined as

H(q,p) = p∂L ∂q⋅ − L

and that the momentum is given by

p = ∂L ∂q⋅ .

Therefore the definition of Legendre transform for a differentiable function is satisfied. □

If a function f(x,y,z,…) is a convex function of several variables, then we may take the Legendre transformation of each variable. In this case, we find a Hamiltonian H(q,p,t). This requires a little more technical work – the convexity condition must be extended to multiple variables, this is a little tricky and is discussed in the problems. However, it remains true, and one may use the same relations to define the Hamiltonian for multiple variables.

H(q,p) = p ⋅∂L ∂q⋅ − L = ∑ i=1Np i ∂L ∂q⋅i − L

p = ∂L ∂q⋅ ⇒pi = ∂L ∂q⋅i

Example. We consider a single particle in a potential, with Lagrangian

L = m 2 q⋅2 − V (q)

the momentum is

p = ∂L ∂q⋅ = mq⋅,

so we can rewrite the Lagrangian in terms of p as

L = 1 2m p2 − V (q).

Then we find our Hamiltonian as expected

H = 1 2m p2 + V (q)

Example. General quadratic energy. Suppose M is an N × N symmetric matrix and our system is given by the Lagrangian (no potential term)

L = 1 2q⋅ ⋅ M ⋅q⋅ = 1 2∑ i=1N ∑ j=1Nq⋅ iMijq⋅j.

Then the momentum is given by

p = M ⋅q⋅,pi = ∑ jMijq⋅j⇒q⋅ = M−1 ⋅p

so that the Hamiltonian becomes

H = 1 2p ⋅ M−1 ⋅p.

The Hamiltonian only exists if M is invertible, so that detM≠0. In fact, M must be positive definite so all eigenvalues must be positive. For systems which do not satisfy this condition, there is no Hamiltonian description, even if there is a Lagrangian one.
