2.3 Constraints in Lagrangian mechanics

The pendulum systems we considered above had an implicit constraint that the pendulum length was fixed. Sometimes, however it is not possible to incorporate the constraints in such a natural way.

Definition. A constraint on a mechanical system with generalised coordinates q is called holonomic if it can be written as

g(q,t) = 0

for some function g.

Note that holonomic constrains depend only on the generalised coordinates, not the velocities. There is a further distinction on holonomic constraints between scleronomic where g has no explicit time-dependence and rheonomic where it does.

Definition. A constraint on a mechanical system with generalised coordinates q is called non-holonomic if it can be written as

f(q,q⋅,t) = 0

for some function f with explicit dependence on q⋅.
