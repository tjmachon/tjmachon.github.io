3.2 Hamilton’s equations and phase space

In Lagrangian mechanics, our system was described by a path q(t) in the configuration space of the system. For a system with N degrees of freedom, the configuration space was of dimension N. the quantity q⋅ arose only as the velocity of a trajectory. In Hamiltonian mechanics the momenta p are promoted to full independent variables, equal but different to the position variables q. We then obtain a 2N dimensional space, spanned by the N q and p values. This is called phase space.

Definition. The phase space of a mechanical system described by N generalised coordinates q, is the 2N dimensional space given by (q,p).

Proposition. Hamilton’s equations. The Euler-Lagrange equations for a mechanical system with Lagrangian L(q,q⋅,t) and Hamiltonian H(q,p,t) are equivalent to Hamilton’s equations

dqi dt = ∂H ∂pi ,dpi dt = −∂H ∂qi

on phase space

Proof. For simplicity we work with one coordinate, q. The extension follows naturally. We use the definition of Hamiltonian H = pq⋅ − L(q,q⋅,t), then we find Hamilton’s first equation is

dq dt = ∂H ∂p = q⋅,

Hamilton’s second equation gives

dp dt = −∂H ∂q = ∂L ∂q

recall that since the Hamiltonian is the Legendre transform of L, p = ∂L ∂q⋅ , so the second equation is rewritten as

d dt ∂L ∂q⋅ = ∂L ∂q ,

these are just the Euler-Lagrange equations. □

If the Hamiltonian does not depend explicitly on time, then it is conserved.

Proposition. In a Hamiltonian system with Hamiltonian H(q,p,t) we have

dH dt = ∂H ∂t

Proof.

dH dt = ∂H ∂q ∂q ∂t + ∂H ∂p ∂p ∂t + ∂H ∂t = ∂H ∂q ∂H ∂p −∂H ∂p ∂H ∂q + ∂H ∂t .

□

Hamilton’s equations can be recast in matrix form. In 2D this is written as

d dt q p = 0 1 −1 0 ∂H ∂q ∂H ∂p = Ω⋅∇H

For N dimensions, they can be written as

d dt q p = 0 I −I 0 ∂H ∂q ∂H ∂p = ΩN⋅∇H

where I corresponds to an N × N identity matrix. ΩN is sometimes known as the symplectic form.

Example. Hamiltonian description of simple harmonic motion. Consider a one-dimensional particle of mass m in a harmonic potential V (q) = kq2∕2. The Lagrangian is

L = m 2 q⋅2 −k 2q2

The momentum is p = mq⋅ and we find the Hamiltonian

H = 1 2mp2 + k 2q2

Hamilton’s equations then give

q⋅ = p m,p⋅ = −kq

With solution

q = Acos(ωt + ϕ),p = −Amωsin(ωt + ϕ),ω = k∕m

Sketching the dynamics in phase space, we see ellipses (as in Figure 27 from the next example). In fact, each trajectory in phase space follows the ellipse

1 2mp2 + k 2q2 = const.

given by finding the contour of fixed energy in phase space. The aspect ratio (or eccentricity) of the ellipse is fixed by k and m, the size by the total energy which varies from trajectory to trajectory.

Example. Hamiltonian description of a pendulum.

The Lagrangian is

L = m 2 L2𝜃⋅2 − mgL(1 − cos𝜃)

so the Hamiltonian is given by

H(𝜃,p) = p𝜃⋅ −m 2 L2𝜃⋅2 + mgL(1 − cos𝜃)

with the momentum p given by

p = ∂L ∂𝜃⋅ = mL2𝜃⋅

rearranging we find

H(𝜃,p) = p2 mL2 −1 2 p2 mL2 + mgL(1 − cos𝜃) = 1 2 p2 mL2 + mgL(1 − cos𝜃)

as a function on the two-dimensional phase space (𝜃,p). Let’s first assume the small-angle approximation, where 1 − cos𝜃 ≈ 𝜃2∕2, then we have

H(𝜃,p) = 1 2mL2p2 + mgL 2 𝜃2

Hamilton’s equations then read

∂𝜃 ∂t = p mL,

Which we can solve to give

𝜃(t) = Acos(ωt + ϕ)

p(t) = −mωLAsin(ωt + ϕ)

with ω2 = g∕L. It is, however, much more instructive to look at the phase-space behaviour of the system. We can plot the level sets of the Hamiltonian, which are also the phase space trajectories, we find these are ellipses!

The full pendulum equations are difficult to solve. The Hamiltonian perspective, however, allows us to understand the qualitative behaviour very easily. We see that we can obtain a qualitative picture of the pendulum’s behaviour simply by studying the topology of the trajectories.

Example. Hamiltonian for relativistic particles. The Lagrangian for a free relativistic particle was given by

L = −mc2 γ = −mc21 − |v |2 ∕c2

which was expressed in terms of the proper time. Using this we find the conjugate momentum

p = ∂L ∂v = mγv,γ = 1 1 − |v |2 ∕c2.

This implies

γ = 1 + |p |2 ∕m2 c2

so that the Hamiltonian is

H = p ⋅v − L = mγc2,

which is the relativistic energy of a free particle. We can then use Hamilton’s equations to find the motion of the particle

q⋅ = v = ∂H ∂p = p mγ,p⋅ = −∂H ∂q = 0.

So when there is no force, the relativistic momentum of a particle is conserved. The relativistic energy is also conserved, since H is constant. This is reference-frame dependent, but is constant within each reference frame.

Example. Hamiltonian for charged particles. The Lagrangian for a charged particle was given by

L = m 2 r⋅2 − eϕ + er⋅ ⋅A

This implies the caonical momentum is

p = mr⋅ + eA

note that the canonical momentum is not gauge invariant. We then find the Hamiltonian is given by

H = 1 2m(p − eA)2 + eϕ.

Note that in contrast with the Lagrangian, the Hamiltonian is gauge invariant. The term p − eA is the kinetic momentum.

Example. Ignorable coordinates. We study the motion of a particle of mass m in three dimensions constrained to lie on the surface of a cylinder of radius R inside a central potential V = k(x2 + y2 + z2)∕2 (see Figure 29. Using cylindrical coordinates

We can now incorporate the constraint R = r, by defining our generalised coordinates q = (𝜃,z), in terms of which our Lagrangian becomes

L = m 2 (R2𝜃⋅2 + z2) −k 2(R2 + z2)

The Hamiltonian approach gives

H = 1 2m p𝜃2 R2 + pz2 + k 2(R2 + z2)
