3.5 Equilibria

Throughout physics it is important to study equilibrium points, where there is no motion.

Proposition. A point (q,p) in phase space is an equilibrium point of a time-independent Hamiltonian system if and only if the point (q,p) is a critical point of the Hamiltonian, i.e. ∇H = 0 at that point.

Proof. We have

d dt q p = Ω⋅∇H

so

d dt q p 2 = d dt q p ⋅d dt q p = ∇H⋅ΩT ⋅Ω⋅∇H

but the symplectic matrix satisfies ΩT ⋅ Ω = I, the identity matrix, so this becomes

d dt q p 2 = ∇H2

we have an equilibrium point if the LHS is zero, which implies the RHS must also be zero. □

Near an equilibrium point (q0,p0) we can expand the Hamiltonian

H(q0 + δq,p0 + δq) ≈ H(q0,p0) + ∂H ∂q ⋅ δq + ∂H ∂p ⋅ δp

+1 2δp ⋅ ∂2H ∂p∂p ⋅ δp + δq ⋅ ∂2H ∂q∂p ⋅ δp + 1 2δq ⋅ ∂2H ∂q∂q ⋅ δq

where all the derivatives are evaluated at the point (q0,p0). Since we are at an equilibrium point the linear terms vanish, and we find

H(q0 + δq,p0 + δq) ≈ H(q0,p0) + 1 2δp ⋅ ∂2H ∂p∂p ⋅ δp + δq ⋅ ∂2H ∂q∂p ⋅ δp + 1 2δq ⋅ ∂2H ∂q∂q ⋅ δq

or in components

H(q0 + δq,p0 + δq) ≈ H(q0,p0) + ∑ i=1N1 2δpiδpj ∂2H ∂pi∂pj + δqiδpj ∂2H ∂qi∂pj + δqiδqj1 2 ∂2H ∂qi∂qj

We then get two symmetric matrices Hpp and Hqq for the derivatives with respect to p and q, given as

(Hpp)ij = ∂2H ∂pi∂pj,

where again the derivatives are evaluated at (q0,p0). The matrix Hqp is not symmetric, and is defined as

(Hqp)ij = ∂2H ∂qi∂pj.

By rewriting δq↦q we find that near an equilibrium point, any Hamiltonian system can be approximated.

Proposition. Near an equilibrium point a time-independent Hamiltonian system can be approximated by a Hamiltonian system whose Hamiltonian is quadratic in the positions and momenta

d dtqi = ∑ j (Hpp)ijpj + (Hqp)ijT q j

d dtpi = −∑ j (Hqp)ijpj + (Hqq)ijqj

which can be written in matrix form as

d dt q p = Ω⋅∇H,H = 1 2p⋅Hpp⋅p+q⋅Hqp⋅p+1 2q⋅Hqq⋅q.

Proof. Near the equilbrium point we can expand the Hamiltonian to quadratic order. Applying Hamilton’s equations then gives the linear system of equations. □

We can rewrite the linear system as

dX dt = d dt q p = HqpT Hpp −Hqq−Hqp q p = A⋅X

where we have defined the 2N × 2N dimensional matrix A and the 2N-dimensional vector X.

The solution of any linear system like this will by

X = ∑ i=12NX̃ ieλit

where X̃i is the ith eigenvector of A and λi the corresponding ith eigenvalue. Real eigenvalues give exponential decay and growth, imaginary eigenvalues gives oscillations and complex give exponentially growing or decaying oscillations.

Proposition. The eigenvalues of the linear system come in ± pairs.

Proof. Eigenvalues of a matrix A are found by solving det(A − λI) = 0. Applying this to our system we find

det HqpT − λI Hpp −Hqq −Hqp − λI = 0

Now we can rewrite this as

det Ω ⋅ Hqp + λI −Hqq Hpp −HqpT + λI ⋅ Ω = 0

Since det(A ⋅ B ⋅ C) = detAdetBdetC, and detΩ = 1 we find if λ is an eigenvalue then λ also satisfies

det Hqp + λI −Hqq Hpp −HqpT + λI = 0

but this is just the transpose of the original matrix with − λ (recall that Hpp and Hqq are symmetric matrices). Since the transpose does not change the eigenvalues, − λ must also be an eigenvalue. □

Because the matrix is real, any eigenvalues must also be complex conjugates of each other. This means that if λ is an eigenvalue, − λ, λ∗ and − λ∗ are all eigenvalues.

Definition. An equilibrium point of a Hamiltonian system is stable if all the eigenvalues of the linearised system are imaginary.

A system is stable if there are no exponentially growing modes. But the eigenvalue symmetry means that all exponentially growing modes come with an exponentially decaying mode. This implies that all the eigenvalues of stable equilibria are imaginary, and hence that they all describe oscillations.

Example. Pendulum equilibria. Let’s return to our pendulum example. The gradient of the Hamiltonian is given by

∇H = p mL2 mgLsin𝜃

we have an equilibrium point if ∇H = 0, hence an equilibrium point (q0,p0) must have

p0 = 0,sin𝜃0 = 0⇒𝜃0 = nπ,ninteger,

looking at the phase space Figure 28. We see that there are two kinds of equilibrium points, stable ones where 𝜃 = 0 or a multiple of 2π, and unstable ones where 𝜃 is an odd multiple of 2π.

Let’s see if this is borne out in the theory. Because we have a two-dimensional phase space, the matrices Hpp, Hqp and Hqq are just numbers. A quick computation shows that

Hpp = 1 L2m,Hqp = 0,Hqq = gLm(−1)n,

where we expand about the equilibrium point with cos𝜃0 = (−1)n.

We therefore have the linearised system

d dt q p = 0 1 L2m gLm(−1)n+1 0 q p

The normal modes of the system are found by computing the eigenvalues of this matrix. We find

λ = ±(−1)n+1 g L

so if n is even (or zero), then the eigenvalues are imaginary – so we get oscillations and the pendulum is stable. These are oscillations about the point where the pendulum points down. If n is odd, then the eigenvalues are real, one is positive and one is negative – and the positive eigenvalue leads to instability. Hence we find that the equilibrium point with the pendulum pointing vertically upwards is unstable.

Example. Canonical transformation for harmonic Hamiltonian. Upon linearising to find a quadratic Hamiltonian, it very often has the form

H = 1 2∑ ijpiMij−1p j + 1 2∑ ijqiKijqj

Any positive-definite real symmetric matrix has a Cholesky decomposition

M = LLT

where L is a ‘lower triangular’ matrix, meaning only elements Lij with i ≥ j are non-zero. Defining a new set of coordinates by the canonical transformation

pj = ∑ kLjkPk′,q j = ∑ kLjk−1Q k′

the Hamiltonian becomes

H = 1 2∑ k(Pk′)2 + 1 2∑ klNklQk′Q l′

where N = (L−1)T ⋅ K ⋅ L−1 is symmetric. Since N is symmetric it can be written as

N = UDNUT

where U is an orthogonal matrix whose columns are eigenvectors of N, and DN is diagonal with entries the eigenvalues of N. Now we make one further canonical transformation

Q = U ⋅Q′,P = UT ⋅P′

and we find

H = 1 2∑ jPj2 + λ jQj2

where λj is the jth eigenvalues of N. We have diagonalised our system into a sum of independent harmonic oscillators (or exponentially decaying / growing modes if the eigenvalues are negative). It is left to the reader to check that these transformations are canonical.
