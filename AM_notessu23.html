3.6 Liouville’s theorem

In this section we consider a time-independent Hamiltonian system. We suppose rather than a single realisation of the system, we have an ensemble (infinitely many copies) of the system, each at a different point in phase space. The function ρ(q,p,t) describes the density of these systems at time t, so that if D is a region of phase space, the integral

Q = ∫ DρdNqdNp

describes the total number of systems, Q, inside the region D at time t. We can think of this as a ‘fluid’ in phase space, each particle of the fluid is a realisation of the Hamiltonian system. The particles move around phase space according to Hamilton’s equations. Now consider our region D, the change in the number of particles will be

dQ dt = −∫ ∂Dv ⋅n̂dA

where n̂ is the unit outward normal to D, dA is the 2N − 1 dimensional ‘hyperarea’ and v = (q⋅,p⋅) is the velocity of the ‘systems’ in phase space. Since the systems in our ensemble are neither created nor destroyed over time, the change in the region D must be equal to the flow through the boundary of D (which we denote ∂D). The minus sign is because we chose the outward normal. But the change in particles must also be

dQ dt = ∫ D∂ρ ∂tdNqdNp

equating these and using the 2N-dimensional divergence theorem we find

0 = ∫ D∂ρ ∂tdNqdNp + ∫ ∂Dv ⋅n̂dA = ∫ D ∂ρ ∂t + ∇⋅ (vρ)dNqdNp

where the 2N-dimensional divergence is given by

∇⋅w = ∑ i=12N∂wi ∂xi

with xi = qi if 1 ≤ i ≤ N and xi = pi−N if N < i ≤ 2N. Since the region D is arbitrary we find

∂ρ ∂t + ∇⋅ (vρ) = 0

Now we know that ∇⋅v = 0, this is because

∇⋅v = ∑ i=1N ∂ ∂qi dqi dt + ∂ ∂pi dpi dt = ∑ i=1N ∂2H ∂qi∂pi − ∂2H ∂pi∂qi = 0

so we may rewrite this equation as

∂ρ ∂t + v ⋅∇ρ = 0

Now we may state Liouville’s theorem

Theorem.

dρ dt = 0

Proof. We have

dρ dt = ∂ρ ∂t + ∑ i=1N ∂ρ ∂qi dqi dt + ∂ρ ∂pi dpi dt

or

dρ dt = ∂ρ ∂t + v ⋅∇ρ = 0

which vanishes by the above calculation. □

Liouville’s theorem has very deep consequences, it states that the flow of a Hamiltonian system preserves volume in phase space. This means if we start with a blob of systems in a region of volume V , at some time later our systems will be a different blob at a different place, but it will still have volume V .
