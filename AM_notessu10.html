2.1 Generalised coordinates

The computational power of Lagrangian mechanics comes through the use of generalised coordinates.

Definition. A set of generalised coordinates q = (q1,q2,…,qN) for a mechanical system completely specifies the configuration of the system.

Example. Suppose we have two particles in three dimensions, their positions are r1 = (x1,y1,z1) and r2 = (x2,y2,z2), then the generalised coordinates are q = (x1,y1,z1,x2,y2,z2) (note this is six-dimensional).

Example. Consider the double pendulum in Figure 12. It is specified by two angles 𝜃1 and 𝜃2 which form generalised coordinates for the system.

One can have any number of different generalised coordinate systems. Often the trick is choosing the right one! If q is one set of generalised coordinates, then another set Q = (Q1,Q2,…,QN) is related to the first by a general transformation

Qi = Qi(t,q1,q2,…,qN).

This implies that the velocities satisfy

Q⋅i = Q⋅i(t,q1,…,qN,q⋅1,…,q⋅N).

An important quantity for a coordinate transformation is the Jacobian.

Definition. The Jacobian J of the transformation q →Q is the N × N matrix

J = ∂Q ∂q ,Jij = ∂Qi ∂qj .

The inverse Jacobian J−1 is given by

Jij−1 = ∂qi ∂Qj.

Example. q = (x,y) are Cartesian coordinates in the plane, and Q = (r,𝜃) polar coordinates. Polar coordinates are determined by

r = x2 + y2,𝜃 = arctany x,

and the Jacobian is equal to

J = ∂r ∂x∂r ∂y ∂𝜃 ∂x∂𝜃 ∂y = 1 x2 + y2 x y −yx

(note the singularity at the origin).

The power of generalised coordinates comes from the following theorem.

Theorem. Let q and Q be two different sets of generalised coordinates for the same mechanical system. If the mechanical system satisfies the Euler-Lagrange equations

∂L ∂q − d dt ∂L ∂q⋅ = 0,

in one set of generalised coordinates, then it also satisfies the Euler-Lagrange equations

∂L ∂Q − d dt ∂L ∂Q⋅ = 0,

in the other set.

Proof. We can rewrite the first term in the Euler-Lagrange equations as

∂L ∂qj = ∑ i=1N ∂L ∂Qi ∂Qi ∂qj + ∑ i=1N ∂L ∂Q⋅i ∂Q⋅i ∂qj

or as a matrix equation

∂L ∂q = JT ⋅ ∂L ∂Q + J⋅T ⋅ ∂L ∂Q⋅

where we have defined the ‘dotted Jacobian’ as

J⋅ij = ∂Q⋅i ∂qj

The second term in the Euler-Lagrange equations can similarly be written as

d dt ∂L ∂⋅qj = d dt ∑ i=1N ∂L ∂Q⋅i ∂Q⋅i ∂q⋅j

(note that the coordinates Qi do not depend on the velocities qj, so terms like ∑ i=1N(∂L∕∂Qi)∂Qi∕∂q⋅j vanish). Now consider the quantity

∂Q⋅i ∂q⋅j

we know that

Q⋅i = ∂Qi ∂t + ∑ k=1N∂Qi ∂qk q⋅k

now ∂Qi∕∂t is a function of t and q only, as is ∂Qi∕∂qk, so we find

∂Q⋅i ∂q⋅j = ∑ k=1N∂Qi ∂qk ∂q⋅k ∂q⋅j = ∂Qi ∂qj

since ∂q⋅k∕∂q⋅j = δkj, 0 if k≠j and 1 if k = j. This allows us to write the second term in the Euler-Lagrange equations as

d dt ∂L ∂⋅q⋅ = d dt JT ⋅ ∂L ∂Q⋅

Putting it together we find

∂L ∂q − d dt ∂L ∂q⋅ = JT ⋅ ∂L ∂Q + J⋅T ⋅ ∂L ∂Q⋅ − d dt JT ⋅ ∂L ∂Q⋅

Now note that the dotted Jacobian is the time derivative of the regular Jacobian J⋅ = dJ∕dt. We can then expand the d∕dt term, get a cancellation and find

∂L ∂q − d dt ∂L ∂⋅q⋅ = JT ⋅∂L ∂Q − d dt ∂L ∂Q⋅

so the system also satisfies the Euler-Lagrange equations with respect to Q. □

With generalised coordinates come generalised forces and momenta.

Definition. For a Lagrangian L(t,q,q⋅) the quantity

Fi = ∂L ∂qi

is the ith generalised force and the quantity

pi = ∂L ∂q⋅i

is the ith generalised (or conjugate, canonical) momentum.
